# Pilot Documentation

This folder contains documentation for specific pilot implementations with different organizations.

---

## Available Pilots

### Attendo

- **[PILOT_PLAN.md](./attendo/PILOT_PLAN.md)** - Attendo pilot plan, objectives, and success criteria
- **[DATA_REQUIREMENTS.md](./attendo/DATA_REQUIREMENTS.md)** - Attendo data requirements (eCare CSV format)

---

## Pilot Structure

Each pilot has its own subfolder containing:

- **PILOT_PLAN.md** - Pilot objectives, timeline, phases, success criteria
- **DATA_REQUIREMENTS.md** - Data format specifications for the pilot organization
- Additional pilot-specific documentation as needed

---

## Adding a New Pilot

1. Create a new folder: `pilots/<organization-name>/`
2. Create `PILOT_PLAN.md` and `DATA_REQUIREMENTS.md`
3. Update this README with the new pilot
