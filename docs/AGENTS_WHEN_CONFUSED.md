# When repo teams or agents are confused

**MEMORY and REPORT are key.** Use them first.

Resolution order — do **1**, then **2**, then **3** only if still unclear:

| Order | Source | What to use |
|-------|--------|-------------|
| **1** | **Repo** | Repo docs, CLAUDE.md, ARCHITECT prompts, code. Single source of truth for that repo. |
| **2** | **DARWIN** | Workspace memory (my/memory/context, decisions, learnings), priorities (my/priorities.md), and **reports** (machine/agent-reports/ — latest-session.md, session-*.md). Shared context across repos. |
| **3** | **Ask me** | Over **Telegram**. Human clarification when repo + DARWIN are not enough. |

Do not guess. Do not invent. 1 → 2 → 3.
