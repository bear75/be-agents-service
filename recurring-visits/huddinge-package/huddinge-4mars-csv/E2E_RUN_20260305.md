## To run solve + metrics (requires TIMEFOLD_API_KEY)

Set the API key, then from `be-agent-service/recurring-visits/scripts/`:

```bash
# 1. Solve (polls until complete; can take several minutes)
export TIMEFOLD_API_KEY="your-key"
# Or use ~/.config/caire/env with line: export TIMEFOLD_API_KEY=...

python3 submit_to_timefold.py solve ../huddinge-package/huddinge-4mars-csv/input/input_4mars_20260305_193758.json \
  --wait --save ../huddinge-package/huddinge-4mars-csv/output
```

When solve finishes, the script prints the output path, e.g.  
`../huddinge-package/huddinge-4mars-csv/output/<route_plan_id>_output.json`.

```bash
# 2. Metrics (use the output file from step 1)
python3 metrics.py ../huddinge-package/huddinge-4mars-csv/output/<route_plan_id>_output.json \
  --input ../huddinge-package/huddinge-4mars-csv/input/input_4mars_20260305_193758.json \
  --save ../huddinge-package/huddinge-4mars-csv/metrics
```
