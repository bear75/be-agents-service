> **Last updated:** January 2025

## Overview

This guide describes how to configure sitemap and robots.txt for SEO optimization.

## Sitemap Generation

### vite-plugin-sitemap

#### Installation

```bash
yarn add -D vite-plugin-sitemap
```

#### Configuration

```typescript
// vite.config.ts
import { defineConfig } from "vite";
import react from "@vitejs/plugin-react-swc";
import Sitemap from "vite-plugin-sitemap";

export default defineConfig({
  plugins: [
    react(),
    Sitemap({
      hostname: "https://hemtjanstguide.se",
      dynamicRoutes: [
        "/",
        "/find-home-care",
        "/guides",
        "/guides/apply",
        "/guides/rights",
        "/guides/fees",
        "/about",
        "/contact",
      ],
      changefreq: "weekly",
      priority: 0.8,
      lastmod: new Date(),
      generateRobotsTxt: true,
      robots: [
        {
          userAgent: "*",
          allow: "/",
        },
        {
          userAgent: "GPTBot",
          allow: "/",
        },
      ],
    }),
  ],
});
```

## robots.txt

### Standard Template

```txt
# robots.txt for [domain]
# Generated by vite-plugin-sitemap

User-agent: *
Allow: /

# AI Agents - Explicit Allow
User-agent: GPTBot
Allow: /

User-agent: ChatGPT-User
Allow: /

User-agent: anthropic-ai
Allow: /

User-agent: Claude-Web
Allow: /

User-agent: PerplexityBot
Allow: /

User-agent: CCBot
Allow: /

User-agent: Google-Extended
Allow: /

# Block admin/preview paths (if applicable)
User-agent: *
Disallow: /admin/
Disallow: /preview/
Disallow: /api/

# Sitemap
Sitemap: https://[domain]/sitemap.xml
```

### Location

The file should be in `public/robots.txt`:

```
apps/hemtjanstguide/
├── public/
│   ├── robots.txt
│   └── sitemap.xml  # Generated at build
```

## Manual Sitemap (if plugin not used)

### Static sitemap.xml

```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
```

## Dynamic Sitemap for Data-Driven Pages

### Script to generate sitemap

```typescript
// scripts/generate-sitemap.ts
import fs from "fs";

interface SitemapEntry {
  loc: string;
  lastmod: string;
  changefreq: string;
  priority: number;
}

async function generateSitemap() {
  const hostname = "https://hemtjanstguide.se";

  // Static routes
  const staticRoutes: SitemapEntry[] = [
    {
      loc: "/",
      lastmod: new Date().toISOString().split("T")[0],
      changefreq: "weekly",
      priority: 1.0,
    },
    {
      loc: "/guider",
      lastmod: new Date().toISOString().split("T")[0],
      changefreq: "weekly",
      priority: 0.8,
    },
  ];

  // Dynamic routes (fetch from API/database)
  // const municipalities = await fetchMunicipalities();
  // const dynamicRoutes = municipalities.map(m => ({
  //   loc: `/kommun/${m.slug}`,
  //   lastmod: m.updatedAt,
  //   changefreq: "weekly",
  //   priority: 0.7,
  // }));

  const allRoutes = [...staticRoutes];

  const xml = `<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
${allRoutes
  .map(
    (r) => `  <url>
    <loc>${hostname}${r.loc}</loc>
    <lastmod>${r.lastmod}</lastmod>
    <changefreq>${r.changefreq}</changefreq>
    <priority>${r.priority}</priority>
  </url>`,
  )
  .join("\n")}
</urlset>`;

  fs.writeFileSync("public/sitemap.xml", xml);
  console.log("Sitemap generated!");
}

generateSitemap();
```

### package.json script

```json
{
  "scripts": {
    "sitemap": "tsx scripts/generate-sitemap.ts",
    "prebuild": "npm run sitemap"
  }
}
```

## IndexNow Integration

### What is IndexNow?

IndexNow is a protocol for notifying search engines (Bing, Yandex) about updates immediately, instead of waiting for crawl.

### Setup

1. Generate API key (UUID)
2. Place key file in public/

```bash
# Generate key
uuidgen > public/[key].txt
```

### Submit URLs

```typescript
// lib/indexnow.ts
const INDEXNOW_KEY = "your-api-key";

async function submitToIndexNow(urls: string[]) {
  const response = await fetch("https://api.indexnow.org/indexnow", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({
      host: "hemtjanstguide.se",
      key: INDEXNOW_KEY,
      keyLocation: `https://hemtjanstguide.se/${INDEXNOW_KEY}.txt`,
      urlList: urls,
    }),
  });

  return response.ok;
}
```

## Google Search Console

### Verify Domain

1. Go to [Google Search Console](https://search.google.com/search-console)
2. Add property for each domain
3. Verify via DNS or HTML file

### Submit Sitemap

1. In Search Console, go to "Sitemaps"
2. Enter URL: `https://example.com/sitemap.xml`
3. Click "Submit"

### Monitor

- Coverage (indexed pages)
- Errors (404, redirect loops)
- Core Web Vitals
- Mobile usability
